# speechtotext

Local privacy-preserving speech-to-text with speaker diarization.

See `docs/superpowers/specs/2026-05-14-speech-to-text-cli-design.md` for design.

## Install (dev)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

## Run tests

```bash
pytest
```
